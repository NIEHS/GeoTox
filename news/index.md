# Changelog

## GeoTox (development version)

- Fixed a bug that affected sensitivity analysis. The column order of
  C_ss values used for sensitivity analysis is now explicitly ordered
  similar to other functions.

- Added set_population() function.

- The simulate\_\* functions can now handle population sizes that vary
  across regions.
