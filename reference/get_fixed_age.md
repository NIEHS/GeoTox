# Get `C_ss` Data for Fixed Age

Get `C_ss` Data for Fixed Age

## Usage

``` r
get_fixed_age(simulated_css, age)
```

## Arguments

- simulated_css:

  list of pre-generated `C_ss` data, for details see:
  [`vignette("package_data", package = "GeoTox")`](https://github.com/NIEHS/GeoTox/articles/package_data.md).

- age:

  list of atomic vectors containing ages.

## Value

list of matrices containing median `C_ss` values.
